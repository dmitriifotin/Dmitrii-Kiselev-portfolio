An analog load cell scale

Implements the functionality to switch between metric/imperial and set the weight of a current object as zero, to allow measuring non-solids. Audio/visual indicators for weights over the limit of 5 kg.

MCU: Arduino Uno
Script: C++
Components: Load cell, LCD display, potentiometer, Piezo speaker
Timeline: 1 month
Team role: circuit design (collab), code (full)
